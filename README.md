  1. Abrir una terminal de linux en la ubicación del proyecto.
  2. Escribir los siguientes comandos:
     1.1. cmake .
     1.2. make
     1.3. ./TrabajoParalela <ubicación archivo .xlsx> <ubicación archivo .csv>
